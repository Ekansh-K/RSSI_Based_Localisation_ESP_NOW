Path-loss calibration

Calibrate one anchor at a time at its final mounted position. Results save to the layout config automatically.

What you need

  • Tape measure + floor marks every 1 m, starting at 1 m from the anchor (e.g. 1 m, 2 m, 3 m, …)
  • Tag powered and broadcasting (flash env tag)
  • This anchor powered on Wi‑Fi; ideally flashed with anchorN_cal (calibration mode)
  • Clear line-of-sight — no people/furniture between tag and anchor
  • Tag at same height as anchor, antenna vertical
  • Do not move the anchor after calibrating
  • Do not run live tracking expectations during cal — finish cal first

Target RMSE < 3 dB (excellent), < 5 dB acceptable. Recollect noisy distances if needed.

How it works

  1. Select anchor ID → Start session
  2. Place tag at next 1 m step → Collect samples (~80)
  3. Commit point → repeat for ≥3 distances (use 1 m spacing)
  4. Fit model → A (RSSI @ 1 m) and n (path-loss exponent) saved
  5. Reflash anchor to normal mode (anchorN) for live use
